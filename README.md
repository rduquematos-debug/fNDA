## About This Project

**Every line of code in this repository was generated by [Claude](https://claude.ai), an AI assistant by Anthropic, through natural language conversation.**

The person behind this project had:
- Zero prior kernel/driver development experience
- Zero knowledge of macOS IOKit internals
- Zero exposure to NVIDIA GPU architecture before starting
- Only a Linux PC with an RTX 3070 Ti, a copy of macOS in a VM, and an unreasonable amount of persistence

**If this sounds impossible вҖ” that's exactly the point.**

The sole technical reference was NVIDIA's [open-gpu-kernel-modules](https://github.com/NVIDIA/open-gpu-kernel-modules) (Linux). No existing macOS NVIDIA driver was used.

### Boot Flow

```
1. mapBars()          вҶ’ PCI BAR0/BAR1 mapping
2. calculateVramLayout() вҶ’ VRAM heap/ELF/queue layout
3. gspSetupQueues()   вҶ’ CMDQ + MSGQ in shared memory
4. setupRadix3()      вҶ’ Page table for SEC2 DMA
5. bootSEC2()         вҶ’ Boot GSP via security engine
6. GSP boots вҶ’ RPCs  вҶ’ SET_SYSTEM_INFO, ALLOC_ROOT
7. sendGspRpc()       вҶ’ Real GSP communication
8. legacyDisplayInit()вҶ’ Head timing, SOR, window
```
