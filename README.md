## About This Project

**Every line of code in this repository was generated by [Claude](https://claude.ai), an AI assistant by Anthropic, through natural language conversation.**

The person behind this project had:
- Zero prior kernel/driver development experience
- Zero knowledge of macOS IOKit internals
- Zero exposure to NVIDIA GPU architecture before starting
- Only a Linux PC with an RTX 3070 Ti, a copy of macOS in a VM, and an unreasonable amount of persistence

**If this sounds impossible — that's exactly the point.**

The sole technical reference was NVIDIA's [open-gpu-kernel-modules](https://github.com/NVIDIA/open-gpu-kernel-modules) (Linux). No existing macOS NVIDIA driver was used.

## What It Does

| Component | Status |
|-----------|--------|
| PCI BAR mapping via IOPCIDevice | ✅ |
| SEC2 security engine boot | ✅ |
| GSP RISC-V firmware boot | ✅ |
| Message queue protocol (NVIDIA msgq replica) | ✅ |
| GSP RPC communication | ✅ |
| Legacy display engine (1920x1080@60) | ✅ |
| VRAM read/write + triangle rasterization | ✅ |
| VBIOS/DCB display parser | ✅ |
| VFIO/QEMU GPU passthrough | ✅ |
| Firmware patching infrastructure | ✅ |

## Architecture

```
┌──────────────┐      ┌─────────────────┐      ┌──────────────┐
│  Userspace   │      │   GA104Driver   │      │  NVIDIA RTX  │
│  (App/daemon)│◄────►│  (IOKit kext)   │◄────►│  3070 Ti     │
│              │      │  IOService      │      │  (GA104)     │
└──────────────┘      └────────┬────────┘      └──────┬───────┘
                               │                       │
                      ┌────────┴────────┐     ┌────────┴────────┐
                      │  GSP RISC-V FW  │     │  Legacy Display │
                      │  (cmdq/msgq)    │     │  Engine (MMIO)  │
                      └─────────────────┘     └─────────────────┘
```

### Boot Flow

```
1. mapBars()          → PCI BAR0/BAR1 mapping
2. calculateVramLayout() → VRAM heap/ELF/queue layout
3. gspSetupQueues()   → CMDQ + MSGQ in shared memory
4. setupRadix3()      → Page table for SEC2 DMA
5. bootSEC2()         → Boot GSP via security engine
6. GSP boots → RPCs  → SET_SYSTEM_INFO, ALLOC_ROOT
7. sendGspRpc()       → Real GSP communication
8. legacyDisplayInit()→ Head timing, SOR, window
```

## Build

Requirements: macOS 13+ (Ventura), Xcode CLT.

```bash
cd Src
make compile-test
```

Output: `GA104Driver.kext/Contents/MacOS/GA104Driver` (Mach-O 64-bit kext)
